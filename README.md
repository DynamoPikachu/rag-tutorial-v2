# rag-tutorial-v2
## Setup
### Install venv with libraries
```bash venv/Scripts/Activate ```
### Install dependencies
```bash pip install -r requirements.txt ```
## Run
### Initialise database
```bash python populate_database.py ```
### Use GUI via: ```bash python gui.py ```
### Or call script directly via:
```bash python query_data.py "Your prompt here" ```
